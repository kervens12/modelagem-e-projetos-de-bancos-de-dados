# Atividade 01 – Sistema de Condomínio

## 📌 Descrição da Atividade

Uma administradora de condomínios deseja construir um sistema
para gerenciamento de edifícios.
É importante para a administradora saber o nome e o endereço de
cada condomínios administrado por ela.
Para cada edifício do condomínio é necessário armazenar a data de
construção, data da última vistoria e quantidade de apartamento que
cada um possui.
Cada edifício possui vários apartamentos, cada apartamento tem um
número, área (m2) e bloco. Em cada apartamento vivem várias
pessoas, as pessoas possuem nome, CPF, data de nascimento e
sexo.

---

## 🎯 Objetivo

Aplicar os conceitos de modelagem conceitual e lógica de banco de dados para representar corretamente as entidades, atributos e relacionamentos descritos no enunciado.

---

## 📂 Artefatos Desenvolvidos

- Modelo Conceitual (MER)
- Diagrama Entidade-Relacionamento (DER)
- Modelo Lógico
